📌 Online Exam Cheating Detection System

📖 Project Overview

Online Exam Cheating Detection System ek AI-based web application hai jo online exams ke dauran students ki suspicious activities detect karta hai. Yeh system face detection, multiple face detection, tab switching aur unusual movements ko track karke cheating prevent karne mein help karta hai.

🎯 Objective

Is project ka main objective hai:

Online exams ko fair banana

Cheating activities ko detect karna

Automated monitoring system develop karna

Manual invigilation ka load kam karna

⚙️ Features

👤 Face Detection using OpenCV

👥 Multiple Face Detection Alert

👀 Real-time webcam monitoring

🔄 Tab switching detection (if implemented)

🚨 Warning system for suspicious activity

🧠 AI-based monitoring logic

🛠️ Technologies Used

Python 🐍

OpenCV (Face Detection)

Flask (Web Framework)

HTML, CSS (Frontend)

JavaScript (Basic interactivity)
